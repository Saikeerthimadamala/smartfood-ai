# AI-Powered Food Waste Reduction System

## Problem
Restaurants waste food due to poor demand prediction.

## Solution
The system predicts how much food to prepare using past data and intelligent estimation logic.

## Tech Stack
- React
- Spring Boot
- MySQL
- Java

## Features
- Demand prediction
- Dashboard
- Data input
- Simple AI-based estimation logic

## Market Crash Handling
If the prediction system fails:
- Switch to rule-based estimation
- Provide manual override option
- Ensure data backup to prevent loss

## AI Approach
Currently, the prediction is implemented using a rule-based logic in Java (Spring Boot), where past sales data is used to estimate future demand.

This simulates an AI-based system and can be upgraded to a real Machine Learning model in the future.
